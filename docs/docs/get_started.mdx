## Quick Start

Get up and running with Rippletide in three steps: install the CLI, point it at your agent, and see evaluation results.

### 1. Install the CLI

```bash
npm install -g rippletide
```

Or run directly with npx:

```bash
npx rippletide
```

### 2. Prepare Your Test Data

Create a `qanda.json` file with the questions and expected answers you want to test:

```json
[
  {
    "question": "What is your return policy?",
    "answer": "We offer a 30-day return policy for most items."
  },
  {
    "question": "How can I track my order?",
    "answer": "Log into your account and go to the My Orders section."
  }
]
```

### 3. Run Your First Evaluation

```bash
rippletide
```

The CLI will prompt you for your agent's endpoint URL, then automatically:
- Send each question to your agent
- Compare responses against expected answers
- Detect hallucinations
- Display a pass/fail summary with a link to the detailed dashboard
