## Get Help

- **Discord** - Join our [Discord community](https://discord.gg/zUPTRH5eFv) for quick questions, feature discussions, and voice calls with the team.
- **GitHub Issues** - [Report bugs or request features](https://github.com/rippletideco/starter/issues) on our GitHub repository.
- **Email** - Reach us at [contact@rippletide.com](mailto:contact@rippletide.com) for partnership or enterprise inquiries.

## Contribute

Rippletide is open source. Contributions are welcome:

1. Fork the [starter repository](https://github.com/rippletideco/starter)
2. Read the [Contributing Guide](https://github.com/rippletideco/starter/blob/main/CONTRIBUTING.md)
3. Open a pull request

## Feedback

We actively use feedback and feature requests to shape the product roadmap. The fastest way to influence what we build next is to tell us what you need on [Discord](https://discord.gg/zUPTRH5eFv) or [GitHub](https://github.com/rippletideco/starter/issues).
