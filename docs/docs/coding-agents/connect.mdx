## Setup

The CLI tool `rippletide-code` handles all the wiring. One command connects Claude Code to a Context Graph:

```bash
npx rippletide-code@latest connect
```

This configures Claude Code hooks and generates the agent instruction files so Claude Code knows how to use the Context Graph.

### What gets generated

After running `connect`, your project will contain:

```
your-project/
├── .claude/settings.json  # Claude Code hooks configuration
└── CLAUDE.md              # Agent instructions (auto-generated)
```

## Data privacy

Rippletide only relies on the context available inside your local Claude Code workflow to identify reusable engineering rules:

- the current Claude Code chat session for the active project
- your `CLAUDE.md`

Your codebase is analyzed through your own local Claude Code environment, not centrally ingested as a full codebase.

Rippletide only stores the extracted rules and conventions that should persist across sessions, not the rest of the project content or unrelated session context.
