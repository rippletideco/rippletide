## Hosted Server (Recommended)

The hosted MCP server at `mcp.rippletide.com` requires no configuration beyond your agent ID:

```json
{
  "mcpServers": {
    "rippletide-kg": {
      "type": "url",
      "url": "https://mcp.rippletide.com/mcp?agentId=your-agent-id"
    }
  }
}
```

### Agent ID

Each agent ID gets its own **isolated knowledge graph**. You can find your agent ID on the agent detail page in the [Rippletide platform](https://eval.rippletide.com).

You can also switch agents at runtime using the `switch_agent` tool during a conversation.

## Self-Hosted Server

If you're running the MCP server yourself (see [Deploy](/docs/mcp/deploy)), you can configure it via environment variables or CLI flags. **CLI flags take priority.**

| Env Variable | CLI Flag | Default | Description |
|-------------|----------|---------|-------------|
| `GRAPH_API_URL` | `--api-url` | `http://localhost:3000` | Knowledge graph backend URL |
| `AGENT_ID` | `--agent-id` | `"default"` | Target agent's graph |
| `TRANSPORT` | `--transport` | `stdio` | `stdio` for local, `http` for remote |
| `PORT` | `--port` | `8080` | Port for HTTP transport |
| `HOST` | `--host` | `127.0.0.1` | Bind address for HTTP transport |
| `LOG_LEVEL` | `--log-level` | `info` | `debug`, `info`, `warn`, `error` |

### Transport Modes

**stdio (default)** — Used by local MCP clients like Cursor and Claude Desktop. The AI client spawns the MCP server as a subprocess.

```json
{
  "mcpServers": {
    "rippletide": {
      "command": "npx",
      "args": ["-y", "@rippletide/mcp"],
      "env": {
        "GRAPH_API_URL": "http://localhost:3000",
        "AGENT_ID": "your-agent-id"
      }
    }
  }
}
```

**HTTP** — Used when deploying as a standalone service:

```bash
npx @rippletide/mcp --transport http --host 0.0.0.0 --port 8080 --api-url https://your-backend.com
```

### Health Check

On startup, the server pings `GET /api/graph/stats` on the backend. If unreachable, a warning is logged but the server still starts.

### Logging

All logs are JSON-formatted and written to stderr (stdout is reserved for the MCP stdio protocol).

```bash
LOG_LEVEL=debug npx @rippletide/mcp
```
