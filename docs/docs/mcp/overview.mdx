## What is the MCP Server?

The Rippletide MCP server connects AI clients (Cursor, Claude Desktop, Claude Code) to your agent's knowledge graph. It lets your AI assistant **remember**, **recall**, and **reason** over structured knowledge and across conversations to reduce unpredictability.

## Quick Start

Add this to your MCP client config:

```json
{
  "mcpServers": {
    "rippletide-kg": {
      "type": "url",
      "url": "https://mcp.rippletide.com/mcp?agentId=your-agent-id"
    }
  }
}
```

Replace `your-agent-id` with your agent ID from the [Rippletide platform](https://eval.rippletide.com).

<CardGroup cols={3}>
  <Card title="Cursor" icon="code">

  </Card>
  <Card title="Claude Desktop" icon="message-bot">
    Add in Claude Desktop MCP settings
  </Card>
  <Card title="Claude Code" icon="terminal">
    Save to `.mcp.json` at your project root
  </Card>
</CardGroup>

That's it. Your AI client now has access to **7 tools** and **4 resources** for persistent memory.

<Info>
  You can find your agent ID on the agent detail page in the [Rippletide platform](https://eval.rippletide.com). Each agent gets its own isolated knowledge graph.
</Info>
